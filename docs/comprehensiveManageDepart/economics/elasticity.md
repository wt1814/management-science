


# 需求弹性之点弹性和弧弹性
&emsp; 某商品的价格为2元／件时，销售量为300件；当价格提高到4元／件时，销售量为100件。按照弧弹性公式计算，求该商品的需求价格弹性。  

&emsp; 解析：考核弧弹性的计算公式。    
1. ΔQ=100-300=-200  
2. ΔP=4-2=2  
3. Q1+Q2）/2=（300+100）/2=200  
4. （P1+P2）/2=（2+4）/2=3  
5. 需求价格弹性=[ -200/200]/（2/3）=-1.5。  

&emsp; 由于需求价格弹性系数始终是负数，所以可以直接用绝对值表示。
